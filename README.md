#  Appointment Management System

## Overview

The Medical Appointment Management System is a modern web application designed to streamline the process of scheduling and managing healthcare appointments. The platform provides an intuitive interface for patients to book appointments and view doctor information while ensuring a seamless user experience.

## Features

### Patient Features

* Book appointments online
* View available doctors
* Select preferred appointment dates
* User-friendly and responsive interface
* Form validation for appointment requests

### Dashboard Features

* Doctor listing and profiles
* Appointment management
* Responsive design for desktop and mobile devices
* Modern and clean user interface

## Technology Stack

### Frontend

* React.js
* JavaScript (ES6+)
* HTML5
* CSS3

* Visual Studio Code

## Installation

1. Clone the repository

```bash
git clone <https://github.com/affordmedtest/Campus-Evaluation-FE>
```

2. Navigate to the project directory

```bash
cd project-folder
```

3. Install dependencies

```bash
npm install
```

4. Start the development server

```bash
npm start
```
